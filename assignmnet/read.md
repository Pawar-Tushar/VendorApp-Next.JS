# Vendor Management App

Built with:
- Next.js 15 (App Router)
- TypeScript
- MongoDB
- TailwindCSS
- Google Auth (NextAuth.js)

## Features
- Google Login / Logout
- Create / Edit / Delete Vendors
- Protected routes
- Responsive UI

## Tech Stack
- Next.js
- NextAuth.js
- Mongoose
- MongoDB Atlas
- TailwindCSS

## How to Run Locally
1. Clone the repo
2. Create `.env.local`
3. Run `npm install`
4. Run `npm run dev`
